About Cloud Cast

Kenya weather updates with a simple city-first experience.

Cloud Cast was designed to make local weather easier to read for four key Kenyan cities: Nairobi, Eldoret, Kisumu, and Nakuru.

What the site includes

City-based weather loading for Nairobi, Eldoret, Kisumu, and Nakuru.

Current temperature and condition indicators such as clear, cloudy, rainy, or stormy.

Short-term forecast cards for the next 5 days.

Simple alert messaging based on rainfall probability in the forecast.

How the data works

The homepage uses fixed coordinates for the selected cities and requests forecast data from Open-Meteo.

API References
