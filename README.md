# Cloud Cast

A simple, clean weather app that helps you stay one step ahead of the sky.

# Overview
Cloud Cast is a lightweight web app that provides real-time weather updates for any location. Whether you're planning your day or checking if you need an umbrella, Cloud Cast delivers accurate and easy-to-read forecasts.

# Features
-  Search weather by city or location  
-  Current temperature, humidity, and conditions  
-  Short-term forecast  
-  Fast and responsive design  
-  Mobile-friendly interface  

# Getting Started
1. Open the app in your browser  
2. Enter a city name in the search bar  
3. View the current weather and forecast instantly  

# Tech Stack
- HTML, CSS, JavaScript  
- Weather API integration  

#  Future Improvements
- 7-day extended forecast  
- Location auto-detection  
- Dark mode toggle  
- Weather alerts and notifications
